Execution Context:
  $ec = <rb_execution_context_t *@...>
  VM Stack: <VALUE *@...> size=...
  $cfp = <rb_control_frame_t *@...>
  Exception: None

Convenience variables set:
  $ec      - Execution context
  $cfp     - Control frame pointer

Now you can use:
  rb-object-print $errinfo
  rb-object-print $ec->cfp->sp[-1]
  rb-stack-trace
